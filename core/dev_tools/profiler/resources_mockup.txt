 USAGE         buffer usage flags | BDA
 SIZE log/cap  buf.size (logical) vs _next_power_of_2 size_class
 WASTE         cap - logical (bucketing internal fragmentation)
 LIFE          first_use-last_use (node index range)
 LIFE cells    per-node W/r from ImageAccess.is_write; ./= from first/last_use
 PROD          producer node index -> nodes[producer].pass->name
 R / W         read / written bools

   Peak concurrent     interval-overlap pass over first_use/last_use
   Aliasing candidates interval-overlap test; pairs with disjoint lifetimes

 CORRECTNESS NOTE:

   POST-cull execution order (map through the culled set) or the LIFE bars
   will not line up with what actually ran.
